22 void QgsGeometryDuplicateNodesCheck::collectErrors( const QMap<QString, QgsFeaturePool *> &featurePools, QList<QgsGeometryCheckError *> &errors, QStringList &messages, QgsFeedback *feedback, const LayerFeatureIds &ids ) const
23 {
24  Q_UNUSED( messages )
25 
26  QMap<QString, QgsFeatureIds> featureIds = ids.isEmpty() ? allLayerFeatureIds( featurePools ) : ids.toMap();
27  QgsGeometryCheckerUtils::LayerFeatures layerFeatures( featurePools, featureIds, compatibleGeometryTypes(), feedback, mContext );
28  for ( const QgsGeometryCheckerUtils::LayerFeature &layerFeature : layerFeatures )
29  {
30  const QgsAbstractGeometry *geom = layerFeature.geometry().constGet();
31  for ( int iPart = 0, nParts = geom->partCount(); iPart < nParts; ++iPart )
32  {
33  for ( int iRing = 0, nRings = geom->ringCount( iPart ); iRing < nRings; ++iRing )
34  {
35  int nVerts = QgsGeometryCheckerUtils::polyLineSize( geom, iPart, iRing );
36  if ( nVerts < 2 )
37  continue;
38  for ( int iVert = nVerts - 1, jVert = 0; jVert < nVerts; iVert = jVert++ )
39  {
40  QgsPoint pi = geom->vertexAt( QgsVertexId( iPart, iRing, iVert ) );
41  QgsPoint pj = geom->vertexAt( QgsVertexId( iPart, iRing, jVert ) );
42  if ( QgsGeometryUtils::sqrDistance2D( pi, pj ) < mContext->tolerance )
43  {
44  errors.append( new QgsGeometryCheckError( this, layerFeature, pj, QgsVertexId( iPart, iRing, jVert ) ) );
45  }
46  }
47  }
48  }

52 void QgsGeometryDuplicateNodesCheck::fixError( const QMap<QString, QgsFeaturePool *> &featurePools, QgsGeometryCheckError *error, int method, const QMap<QString, int> & /*mergeAttributeIndices*/, Changes &changes ) const
53 {
54  QgsFeaturePool *featurePool = featurePools[ error->layerId() ];
55  QgsFeature feature;
56  if ( !featurePool->getFeature( error->featureId(), feature ) )
57  {
58  error->setObsolete();
59  return;
60  }
61  QgsGeometry featureGeom = feature.geometry();
62  QgsAbstractGeometry *geom = featureGeom.get();
63  QgsVertexId vidx = error->vidx();
64 
65  // Check if point still exists
66  if ( !vidx.isValid( geom ) )
67  {
68  error->setObsolete();
69  return;
70  }
71 
72  // Check if error still applies
73  int nVerts = QgsGeometryCheckerUtils::polyLineSize( geom, vidx.part, vidx.ring );
74  QgsPoint pi = geom->vertexAt( QgsVertexId( vidx.part, vidx.ring, ( vidx.vertex + nVerts - 1 ) % nVerts ) );
75  QgsPoint pj = geom->vertexAt( error->vidx() );
76  if ( QgsGeometryUtils::sqrDistance2D( pi, pj ) >= mContext->tolerance )
77  {
78  error->setObsolete();
79  return;
80  }
81 
82  // Fix error
83  if ( method == NoChange )
84  {
85  error->setFixed( method );
86  }
87  else if ( method == RemoveDuplicates )
88  {
89  if ( !QgsGeometryCheckerUtils::canDeleteVertex( geom, vidx.part, vidx.ring ) )
90  {
91  error->setFixFailed( tr( "Resulting geometry is degenerate" ) );
92  }
93  else if ( !geom->deleteVertex( error->vidx() ) )
94  {
95  error->setFixFailed( tr( "Failed to delete vertex" ) );
96  }
97  else
98  {
99  feature.setGeometry( featureGeom );
100  featurePool->updateFeature( feature );
101  error->setFixed( method );
102  changes[error->layerId()][error->featureId()].append( Change( ChangeNode, ChangeRemoved, error->vidx() ) );
103  }
104  }
105  else
106  {
107  error->setFixFailed( tr( "Unknown method" ) );
108  }
109 }
